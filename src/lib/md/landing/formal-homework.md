[Typst](https://typst.app) is a typsetting language meant to be a successor to LaTeX. It provides equivalent functionality with simpler syntax and better package management. I found it of great use when typing solutions to my discrete mathematics homework. To make the process simpler, I created this package that provides macros for setting up a document with a title page, questions, and answers. The output is professional and formal, sure to please your professors.

See the package and code at the links below:
